1. Перевыпусти токен через @BotFather
2. Вставь новый токен в config.py
3. Вставь свой username менеджера
4. Установи зависимости:
   pip install -r requirements.txt

5. Запуск сервера:
   uvicorn server:app --reload

6. Запуск бота:
   python bot.py

ВАЖНО:
Telegram Web App работает только через HTTPS.
